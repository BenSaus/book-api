# Book API

A Severless GraphQL API made with Typescript

## Technologies

Project is created with:

-   Typescript
-   Node JS
-   AWS Lambda
-   GraphQL
-   PostgreSQL (AWS RDS)
-   Docker

## Setup

To run this project, install and run it locally using npm:

```
$ git clone https://github.com/BenSaus/book-api
$ npm run docker
```
